# 워크트리에서 작업 시작

새로운 git worktree를 생성하고 해당 디렉토리로 이동하여 작업을 준비합니다.

## 실행 순서

1. `EnterWorktree` 도구를 사용하여 새 워크트리를 생성합니다.
   - 사용자가 이름을 지정한 경우: `$ARGUMENTS`를 워크트리 이름으로 사용
   - 이름이 없으면 자동 생성
2. 워크트리 생성 후 `Bash` 도구로 해당 워크트리 경로로 `cd`합니다.
3. 경로와 브랜치 이름을 사용자에게 알려줍니다.
4. 작업할 내용을 사용자에게 물어봅니다.
